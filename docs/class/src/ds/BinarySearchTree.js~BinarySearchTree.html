    BinarySearchTree

    A binary search tree is a container where each value is stored in a node, and each node has two children: a left and a right. Values less than the value of the node are put in the left child, and values greater than it are put in the right child. In general, this results in O(log n) search, insertion, and deletion, for only O(n) space.
